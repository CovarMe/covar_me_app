Welcome to CovarMe

The database currently tracks {{ tickers_total }} stocks out of which {{ tickers_available }} are currently fully up-to-date and available for calculations


To serve the estimated covariance values more quickly, the application currently stores {{ matrix_items_stored }} individual covariances between companies in a specifically designed matrix database.


The database runs on a seperate server that is connected to the app server through a VPN connection. That other server runs a 'pseudo' Hadoop cluster with one name- and one datanode. The time-series database OpenTSDB runs on top of HBase which runs on the HAdoop distributed file system. Everything is yeld together by YARN and Zookeeper.


This server (the app server) is just a Micro-size AWS instance with 1 GB of Memory and a single-core processor. The database server, however, is a multicore system and houses the MongoDB as well as the Hadoop file system. Unfortunately, the network communication is not 100% stable because the VPN connection isn't perfect. But no server that was able to run the desired software was affordable.
